# Loan Default Risk Analysis

## Overview
Analysis of 32,576 bank customers to identify key factors 
driving loan defaults using Python and Power BI.

## Key Findings
- 21% of customers defaulted on their loans
- Low income customers (under 20k) default at 68%
- Grade E loans have 72% default rate
- Renters default at 31% vs homeowners at 6%
- Debt consolidation loans are riskiest at 30%

## Tools Used
- Python (pandas, matplotlib, seaborn)
- Power BI
- Jupyter Notebook

## Files
- `loandefaultanalysis.ipynb` - Python analysis notebook
- `loan_cleaned.csv` - Cleaned dataset
- `loan_default_dashboard.pbix` - Power BI dashboard

## Dataset
32,576 customer records · 12 columns · Source: Kaggle.
